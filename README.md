 # 🌍 TerraGauge
**High-precision environmental impact monitoring.**

TerraGauge is a full-stack dashboard designed to track climate data and visualize environmental shifts in real-time.

---

## 🛠️ Tech Stack
- **Frontend:** Next.js 15 (Turbopack), Tailwind CSS, React
- **Backend:** Python (FastAPI/Flask)
- **Environment:** GitHub Codespaces

## 🚀 Getting Started

### 1. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Accessible at: `http://localhost:3000`

### 2. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```

## 📁 Project Structure
- `/frontend`: Next.js web application and UI components.
- `/backend`: Python API and data processing logic.
- `/docs`: Documentation and architecture notes.
